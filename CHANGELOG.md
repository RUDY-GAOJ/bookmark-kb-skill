# Changelog

## 0.1.0 - 2026-06-08

- Add npm/npx CLI entrypoints: `bookmark-kb-skill`, `bookmark-kb`, and `bookmark-kb-install`.
- Add multi-platform Skill installer for Codex, Claude, Gemini, Cursor, and OpenCode.
- Add local bookmark refresh, search, context bundle, and organization report commands.
- Add Chrome default `Bookmarks` file discovery and `--bookmarks-file` override.
- Add local cache support with `BOOKMARK_KB_HOME`.
